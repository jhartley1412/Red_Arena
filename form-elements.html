Dropdown menus are to be used in an effort allow users to conveniently alter times within the site. Rather than going to another page the user is able to successfully update their desired time with an easy to use dropdown menu. A dropdown menu is also used for the navigation at the mobile size to the web application. While the dropdown menu is activated the users are able to view which item the are selecting with a suttle change to the background color. With the use of #b22426 the site provides noticable distiction without distraction.
