Dropdown menus are to be used in an effort allow users to conveniently alter times within the site. Rather than going to another page the user is able to successfully update their desired time with an easy to use dropdown menu. A dropdown menu is also used for the navigation at the mobile size to the web application. While the dropdown menu is activated the users are able to view which item the are selecting with a suttle change to the background color. With the use of #b22426 the site provides noticable distiction without distraction.

Links

Links should be underlined and colored #419AE7. This is to make page links obvious to the user by providing a classic style for the user to seek out links.

Icons

Icons are to be used in limited number on this site. The arrow icons indicate to a user when there is more informatoin to the right or left of a page.

The Birthday Cake Icon is used to as a playfull denotation of a birthday on the Dashboard newsfeed.

The Calendar Icon is used on the newsfeed to remind employees and volunteers of an upcomint event.

The Leafed Brand is used to show upcoming work anniversaries on the Dashboard Newsfeed.
